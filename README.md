This is a Bluetooth-controlled RC car using Arduino.

## Features
- Wireless control
- Forward And Backward Movement
- Easy Turns control
- Speed control
- Beginner Friendly Setup 

## Components
- Arduino Uno
- Motor Driver(L298-N)
- Bluetooth Module(HC-05)
- Battery(3.7V Li-ion cells)
- Chassis
- Tyres
- Motors
- Connecting wires

## How to Use
1. Upload code
2. Connect circuit
3. Pair Bluetooth
4. Run car
